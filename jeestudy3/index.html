JEE Study #3 : Maven project for Tomcat 7 and Java EE 6

Hi guys !

In this video, we create a Maven web project targeting Servlet 3.0 from Java EE 6, and run this project in Tomcat 7.

About Tomcat versions, servlet versions and Java EE versions

Some theory

Tomcat makes available a document describing which JSRs are targeted by which Tomcat versions.

Basically, Tomcat 7 supports parts of Java EE 6, Tomcat 8.5 supports parts of Java EE 7, and Tomcat 9 supports parts of Java EE 8

Some practice

Let's download the ZIP archive containing Tomcat 7 for Windows x64.

Servlet 3.0 Maven project

Some practice

Let's create a new war maven project.

Then we look for javaee-api in mvnrepository.com and browse the compile depencies to find the corresponding depencency for servlets.

We create a small index.html file as well as a very simple servlet.

Then we create the Tomcat server in Eclipse and deploy the project to it.

It works !

Using Tomcat as a windows service

Some practice

Let's download and run the Windows Service Installer for Tomcat 7.

Now, Tomcat will start automatically, and we need to connect the server in Eclipse to this existing server instead of creating new Tomcat server.
